# Minimalist Gallery

## Features
- Browse images and folders
- View images in full screen
- Rotate images
- Hardware back button support
- Permission handling

## Tech Stack
- Android (Kotlin)
- WebView (HTML, CSS, JavaScript)
- EventBus for IPC

## Installation
1. Clone the repository
2. Open in Android Studio
3. Build and run

## Usage
- Open the app to browse images
- Use hardware back button to navigate back
- Use hardware back button to exit the app
